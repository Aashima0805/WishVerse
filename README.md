# WishVerse 🌌

WishVerse is a cosmic wish-tracking web app where your dreams live as glowing stars in a personal galaxy. Type a wish, watch it fly into the universe, and mark it complete when life delivers — every fulfilled wish explodes into a supernova and lives on forever.

---

## Features

- **Wish Creation** — Type a wish, pick a category, and watch a golden star soar to its place in your galaxy

- **Categories** — Love, Career, Health, Travel, Family, Dream, Other — each with its own star color and glow

- **Wish Deadline** — Optionally set a target date; stars show a live countdown badge and pulse red when overdue

- **Supernova** — Fulfilled wishes are preserved as cards showing when they were wished, fulfilled, and their deadline

- **Wish Analytics** — Live stats for total, completed, remaining, overdue, and completion rate 

- **Shooting Stars & Nebula** — Ambient shooting stars and a slowly shifting nebula background keep the galaxy alive

---

## How To Use

1. Open `index.html` in any modern browser
2. Scroll to **Create A Wish**, choose a category and optionally set a deadline
3. Type your wish and hit ** Send** — watch it fly into the galaxy
4. Click any star to view it, mark it fulfilled, or delete it
5. Head to **Supernova** in the nav to see all your completed wishes
6. Check **Analysis** for your cosmic dashboard

---

## Tech Stack

 Structure  -  HTML5 
 Styling    -  CSS3 (animations, keyframes, glassmorphism, CSS variables) 
 Logic      -  Vanilla JavaScript (ES6+) 
 Drawing    -  Canvas API (constellation lines) 
 Storage    -  localStorage 
 Fonts      -  Google Fonts — Cinzel + Raleway 

No frameworks. No build tools. No backend. Just open and use.
